# Proyecto: Gestión de Tareas (To-Do List) - TDD en Java

## Descripción
Este proyecto es parte de la **Práctica III** de la asignatura *Taller Transversal I* (Curso 25/26) en la **Universidad de La Rioja**. El objetivo es implementar un sistema de gestión de tareas utilizando la metodología **TDD (Test-Driven Development)** y Maven.

## Estructura del Proyecto
El proyecto sigue la estructura estándar de Maven:
- `src/main/java`: Código fuente de la aplicación (Servicio, Repositorio, ToDo).
- `src/test/java`: Pruebas unitarias con JUnit.
- `pom.xml`: Configuración de dependencias y plugins.

## Funcionalidades
- Gestión de tareas (CRUD).
- Agenda de contactos por email.
- Sistema de alertas automáticas para tareas fuera de plazo.
